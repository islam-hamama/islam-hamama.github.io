## Overview

Designed and built a **prototype data logger for an electromagnetic sensor** using the **ESP32 microcontroller**. The device enables low-cost, portable field measurements for geophysical monitoring applications.

## Key Features

- ESP32 microcontroller for wireless connectivity (Wi-Fi / Bluetooth)
- Real-time data acquisition from electromagnetic sensor
- On-board data logging with timestamping
- Compact, low-power design suitable for remote field deployment

## Applications

Intended for use in field campaigns requiring portable electromagnetic monitoring, complementing infrasound and seismic sensor networks.
