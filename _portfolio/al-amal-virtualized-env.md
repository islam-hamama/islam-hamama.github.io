## Overview

As **IT Advisor** (September 2020 – present) for the **Al Amal Program** — a volunteering initiative for fresh Geoscience graduates of Egypt — implemented a fully virtualized computing environment enabling participants to carry out scientific work remotely.

## Implementation

- Configured and managed virtual machines accessible via remote desktop
- Provided fresh graduates access to professional geoscience software (seismology, GIS, data analysis tools) without expensive local hardware
- Designed the network and access-control infrastructure for multi-user remote access

## Impact

- Enabled the digital transformation of the entire program
- Allowed participants across Egypt to collaborate and work on real research data remotely
- Reduced the barrier to entry for fresh graduates with limited computing resources
