**Period:** 2015–2019 and January 2026

Delivered practical training courses for researchers and staff at NRIAG, Egypt, covering:

## Seiscomp System
- Automatic Earthquake Detection System configuration and operation
- Real-time seismic data acquisition and processing
- Network management and station configuration

## Fundamentals of Linux Operating System
- Linux command line and shell scripting
- Server administration and networking basics
- Remote access and data management

## Introduction to Seismology and Infrasound (Acoustics) Technologies
- Principles of seismic wave propagation
- Infrasound monitoring: sensors, arrays, and signal processing
- IMS (International Monitoring System) data formats and tools
